# Obesity Prediction Using Machine Learning

## Project Overview
This project focuses on predicting obesity levels using lifestyle, demographic, and health-related features. The dataset includes information such as age, gender, height, weight, eating habits, physical activity, smoking status, water consumption, transportation mode, and family history.

The goal of this project is to apply data preprocessing, exploratory data analysis, machine learning model training, evaluation, and hyperparameter tuning to classify obesity levels.

## Dataset
The dataset contains 2,111 records and 17 columns.

### Features Used
- Gender
- Age
- Height
- Weight
- Family history with overweight
- Frequent consumption of high-calorie food
- Frequency of vegetable consumption
- Number of main meals
- Consumption of food between meals
- Smoking status
- Daily water consumption
- Calorie monitoring
- Physical activity frequency
- Technology usage time
- Alcohol consumption
- Transportation mode

### Target Variable
- Obesity level

## Project Workflow
1. Loaded the obesity prediction dataset
2. Explored dataset structure and basic information
3. Encoded categorical variables using Label Encoding
4. Scaled numerical features using StandardScaler
5. Performed exploratory data analysis
6. Visualized target distribution and feature relationships
7. Split the dataset into training and testing sets
8. Trained multiple machine learning models
9. Compared model performance using accuracy, precision, recall, and F1-score
10. Performed hyperparameter tuning using GridSearchCV
11. Analyzed feature importance
